## 📱 Описание проекта

Современное Android-приложение «Игра Память» — классическая игра на запоминание и поиск парных карточек, разработанная с использованием принципов Material Design 3.

---

## ✨ Основные возможности

### 🎮 Игровые режимы
- **Пары (2 карточки)** — классический режим поиска пар
- **Тройки (3 карточки)** — усложненный режим с тройками карточек
- **Гибридный режим** — случайное чередование пар и троек

### 📐 Размеры игрового поля
- **4×4** — 16 карточек (8 пар)
- **4×5** — 20 карточек (10 пар)
- **6×6** — 36 карточек (18 пар)

### 📊 Дополнительные функции
- ⏱️ **Таймер** — отслеживание времени прохождения
- 🔢 **Счетчик ходов** — подсчет количества попыток
- 📈 **Статистика** — сохранение рекордов через SharedPreferences
- 🎨 **Анимации** — плавные переходы и flip-эффект карточек
- 🎯 **Рекорды** — отдельная статистика для каждого режима

## 🏗️ Архитектура приложения

### Структура экранов

```
SplashActivity (Заставка)
    ↓
MenuActivity (Главное меню)
    ├── SettingsActivity (Настройки) → MainActivity (Игра)
    ├── StatisticsActivity (Статистика)
    └── AboutActivity (Об авторе)
```

### Основные классы

- **`SplashActivity`** — экран загрузки с анимацией (3 сек)
- **`MenuActivity`** — главное меню с навигацией
- **`SettingsActivity`** — выбор параметров игры
- **`MainActivity`** — игровая активность с логикой
- **`StatisticsActivity`** — отображение статистики
- **`AboutActivity`** — информация об авторе
- **`Card`** — модель карточки (id, изображение, состояние)
- **`GameSettings`** — настройки игры (размер, режим)
- **`GameStatistics`** — работа со статистикой (SharedPreferences)

## 🎮 Как играть

1. **Запуск:** При открытии приложения появляется Splash Screen
2. **Меню:** Выбор опций (начать игру, статистика, об авторе)
3. **Настройки:** Выбор размера поля и режима игры
4. **Игра:**
   - Нажимайте на карточки для открытия
   - Ищите совпадающие пары/тройки
   - Открытые карточки остаются видимыми
   - Не совпавшие карточки переворачиваются обратно
5. **Победа:** После нахождения всех пар открывается диалог с результатами

## 📊 Логика игры

### Режим "Пары" (2 карточки)
- Открывается 2 карточки
- Если совпадают → остаются открытыми (зеленый фон)
- Если нет → закрываются через 1 секунду

### Режим "Тройки" (3 карточки)
- Открывается 3 карточки
- Все три должны совпасть
- Логика аналогична режиму пар

### Гибридный режим
- Случайное определение количества карточек для совпадения (2 или 3)
- Добавляет элемент неожиданности

## 📈 Статистика

Приложение сохраняет:
- ✅ Общее количество игр
- ⏱️ Лучшее время (общее)
- 🔹 Статистика по режиму "Пары"
- 🔺 Статистика по режиму "Тройки"
- 🔀 Статистика по гибридному режиму

Сохранение через `SharedPreferences` — данные не теряются после закрытия приложения.
