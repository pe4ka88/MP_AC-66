## Описание приложения MiniShop

Приложение MiniShop представляет собой простой интернет-магазин с двумя активностями:

### Функциональные возможности:

1. **MainActivity (Список товаров)**
   - Отображение списка товаров с использованием ListView
   - Header с заголовком "My Goods List"
   - Footer с счетчиком выбранных товаров и кнопкой перехода в корзину
   - Каждый товар содержит: ID, название, цену и чекбокс для выбора
   - Динамическое обновление счетчика выбранных товаров
   - Реализован кастомный адаптер ProductAdapter, наследующий BaseAdapter

2. **SecondActivity (Корзина товаров)**
   - Отображение только выбранных товаров
   - Кастомизированный список без чекбоксов
   - Заголовок с количеством товаров в корзине

### Технические особенности:

- **Механизм обратного вызова**: Реализован через интерфейс `OnProductCheckedListener` для оповещения MainActivity об изменении состояния чекбоксов
- **ViewHolder Pattern**: Используется для оптимизации производительности ListView
- **Parcelable**: Класс Product реализует Parcelable для передачи данных между Activity
- **Кастомные адаптеры**: ProductAdapter и CartAdapter, наследующие BaseAdapter

## Требования

- Android Studio Arctic Fox или выше
- Android SDK 21+
- Gradle 8.0+

## Инструкция по запуску

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Подключите эмулятор или реальное устройство Android
4. Нажмите кнопку Run (▶) или используйте Shift+F10
5. Выберите устройство для запуска приложения

## Использование приложения

1. При запуске открывается список из 25 товаров
2. Выберите товары, отметив их чекбоксами
3. Счетчик внизу экрана показывает количество выбранных товаров
4. Нажмите кнопку "SHOW CHECKED ITEMS" для перехода в корзину
5. В корзине отобразятся только выбранные товары
6. Используйте кнопку "Назад" для возврата к списку товаров
