# Игра "Память" на Android

Лабораторная работа №1 - Игра для запоминания пар карточек на Android.

## Описание

Приложение представляет собой классическую игру "Память" (Memory Game), где игрок должен находить пары одинаковых карточек, открывая их по одной.

## Основные функции

### Базовые требования ✅

1. **Игровое поле 4х4** - сетка из 16 карточек (8 пар)
2. **Поочередное открытие карточек** - в каждый момент времени отображается не более одной открытой картинки
3. **Парное удаление** - при совпадении текущей и предыдущей картинки пара удаляется с поля
4. **Перезапуск игры** - кнопка "Новая игра" для начала заново

### Бонусные функции 🌟

1. **Настройки размера поля**
   - Выбор между режимами 4x4 (8 пар) и 6x6 (18 пар)
   - Настройки сохраняются между сеансами игры

2. **Отображение информации**
   - Таймер игры (отображение времени в формате MM:SS)
   - Счетчик ходов
   - Счетчик найденных пар

3. **Статистика**
   - Количество сыгранных игр
   - Лучшее время прохождения
   - Минимальное количество ходов
   - Статистика сохраняется локально через SharedPreferences

4. **Визуальное оформление**
   - Красочный интерфейс с цветными карточками
   - 8 различных геометрических фигур (круги, квадраты, ромбы и т.д.)
   - Анимация открытия карточек
   - Затемнение найденных пар

5. **Диалоги и уведомления**
   - Диалог поздравления при победе с отображением статистики
   - Диалог подтверждения перезапуска игры
   - Диалог просмотра статистики
   - Диалог настроек размера поля

## Структура проекта

### Классы

- **MainActivity.java** - главная активность с логикой игры и UI
- **Card.java** - модель карточки (хранит состояние: открыта, найдена, ID изображения)
- **MemoryGame.java** - игровая логика (проверка совпадений, подсчет ходов, управление состоянием игры)
- **CardAdapter.java** - адаптер для отображения карточек в GridView
- **GameSettings.java** - управление настройками и статистикой через SharedPreferences

### Ресурсы

- **activity_main.xml** - layout главного экрана
- **main_menu.xml** - меню с настройками и статистикой
- **drawable/** - 8 различных карточек + рубашка карты (векторная графика)
  - card_back.xml - синяя рубашка карты
  - card_1.xml - красный круг
  - card_2.xml - зеленый квадрат
  - card_3.xml - желтый ромб
  - card_4.xml - фиолетовый круг
  - card_5.xml - голубой прямоугольник со скругленными углами
  - card_6.xml - розовое кольцо
  - card_7.xml - оранжевый прямоугольник
  - card_8.xml - синий круг

## Как играть

1. Запустите приложение
2. Нажимайте на карточки, чтобы открывать их
3. Запоминайте расположение карточек
4. Находите пары одинаковых изображений
5. Игра заканчивается, когда все пары найдены
6. Используйте меню для настройки размера поля и просмотра статистики

## Технические детали

- **Минимальная версия Android**: API 21 (Android 5.0)
- **Язык программирования**: Java
- **UI компоненты**: GridView, LinearLayout, AlertDialog
- **Хранение данных**: SharedPreferences
- **Асинхронность**: Handler для таймера и задержек

### Игровой процесс

1. Все карточки закрыты в начале игры
2. Игрок открывает первую карточку
3. Открывается вторая карточка
4. При совпадении - пара исчезает (затемняется)
5. При несовпадении - обе карточки закрываются
6. Победа - все пары найдены!
