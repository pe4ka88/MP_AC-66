# Инструкция по запуску игры "Память"

## Требования

- Android Studio (последняя версия)
- Android SDK API 24 или выше
- Эмулятор Android или физическое устройство

## Запуск проекта

### Вариант 1: Через Android Studio

1. Откройте Android Studio
2. Выберите "Open" и откройте папку проекта `lab1`
3. Дождитесь синхронизации Gradle
4. Подключите Android устройство или запустите эмулятор
5. Нажмите на зеленую кнопку "Run" (▶️) или Shift+F10
6. Выберите устройство и дождитесь установки приложения

### Вариант 2: Через командную строку

```bash
cd /Users/senia/AndroidStudioProjects/lab1
./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```

## Возможные проблемы

### Gradle sync failed

- Убедитесь, что у вас установлена последняя версия Android Studio
- Проверьте подключение к интернету для загрузки зависимостей

### Устройство не видно

```bash
adb devices
```

Если устройство не отображается, включите отладку по USB в настройках разработчика

### Ошибки компиляции

- Убедитесь, что установлен Android SDK API 24+
- Проверьте, что все файлы из проекта на месте

## Тестирование

После запуска проверьте:

1. ✅ Отображается поле 4x4 с закрытыми карточками
2. ✅ При нажатии карточка открывается
3. ✅ При совпадении двух карточек они остаются открытыми (затемненными)
4. ✅ При несовпадении карточки закрываются через ~0.6 сек
5. ✅ Таймер работает и отсчитывает время
6. ✅ Счетчик ходов увеличивается
7. ✅ Счетчик пар обновляется при нахождении совпадений
8. ✅ При победе появляется диалог с результатами
9. ✅ Кнопка "Новая игра" перезапускает игру
10. ✅ Меню "Настройки" позволяет изменить размер поля
11. ✅ Меню "Статистика" показывает общую статистику

## Дополнительные возможности

### Изменение размера поля

Нажмите на меню (⋮) в правом верхнем углу → Настройки → Выберите размер

### Просмотр статистики

Нажмите на меню (⋮) → Статистика

### Сброс статистики

Для сброса статистики нужно очистить данные приложения:
Настройки Android → Приложения → Memory Game → Хранилище → Очистить данные

## Контакты и поддержка

При возникновении проблем проверьте:

1. Все ли файлы на месте
2. Правильно ли установлен Android SDK
3. Нет ли ошибок в логах Android Studio (Logcat)
